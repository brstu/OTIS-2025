## Теоретическая часть ##
Структурная схема системы автоматического управления с обратной связью включает:
- **w(t)** — задание (алгоритм функционирования системы);  
- **y(t)** — выходная переменная;  
- **e(t) = w(t) - y(t)** — ошибка управления;  
- **u(t)** — управляющее воздействие;  
- **z(t)** — внешние возмущения.  

Задача регулятора — минимизировать ошибку e(t).  
Идеализированное уравнение ПИД‑регулятора:


\[
u(t) = K \left[ e(t) + \frac{1}{T} \int_0^t e(\tau)\, d\tau + T_D \frac{de(t)}{dt} \right]
\]



Дискретная инкрементная форма:


\[
u(k) - u(k-1) = q_0 e(k) + q_1 e(k-1) + q_2 e(k-2)
\]



где коэффициенты:


\[
q_0 = K \left( 1 + \frac{T_D}{T_0} \right), \quad
q_1 = -K \left( 1 + 2 \frac{T_D}{T_0} - \frac{T_0}{T} \right), \quad
q_2 = K \frac{T_D}{T_0}
\]
