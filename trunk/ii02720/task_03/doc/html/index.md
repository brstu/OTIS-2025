## Структура файлов

### lab3main.cpp
**Главная программа моделирования САУ с ПИД-регулятором**

Программа выполняет симуляцию замкнутой системы автоматического управления

- **runSimulation** (ранее simulateSystem)
  Функция моделирования замкнутой системы управления

- **main**
  Точка входа программы, настройка параметров и вывод результатов

### pid.h / pid.cpp
**Модуль дискретного ПИД-регулятора**

Реализация рекуррентного алгоритма ПИД-регулирования для управления тепловым объектом.

#### Класс `PIDController`

- **Атрибуты:**
  - `K` — коэффициент передачи (Kp)
  - `T` — постоянная интегрирования (Ti)
  - `Td` — постоянная дифференцирования
  - `T0` — период дискретизации
  - `q0, q1, q2` — коэффициенты рекуррентной формулы
  - `prev_error` — буфер предыдущих ошибок [e(k-1), e(k-2)]
  - `prev_output` — предыдущее управление u(k-1)

- **Методы:**
  - `PIDController(double gainK, double integralT, double derivativeTd, double samplingT0)`
    Инициализация регулятора с вычислением коэффициентов q0, q1, q2
    - `gainK` — коэффициент усиления
    - `integralT` — время интегрирования
    - `derivativeTd` — время дифференцирования
    - `samplingT0` — шаг квантования

  - `double calculate(double targetSetpoint, double measuredValue)`
    Расчёт управляющего воздействия по рекуррентной формуле
    - `targetSetpoint` — целевое значение
    - `measuredValue` — измеренное значение
    - **Возвращает:** управляющее воздействие u(k)

  - `void reset()`
    Обнуление состояния регулятора

  - `std::vector<double> getCoefficients() const`
    Получение коэффициентов `[q0, q1, q2]`

  - `std::vector<double> getParameters() const`
    Получение параметров `[K, T, Td, T0]`

### model.h / model.cpp
**Модуль модели теплового объекта управления**

Математическая модель объекта с поддержкой линейного и нелинейного режимов работы.

#### Класс `ProcessModel`

- **Атрибуты:**
  - `params` — вектор параметров [a, b, c, d]
  - `prev_value` — предыдущее значение выхода y(k-1)

- **Методы:**
  - `ProcessModel(const std::vector<double>& modelParameters, double startValue)`
    Создание модели с заданными параметрами
    - `modelParameters` — коэффициенты модели
    - `startValue` — начальная температура

  - `double linearModel(double inputSignal)`
    Линейная модель: y(k) = a·y(k-1) + b·u(k)
    - `inputSignal` — управляющее воздействие
    - **Возвращает:** температура объекта

  - `double nonlinearModel(double inputSignal)`
    Нелинейная модель с квадратичной и тригонометрической составляющими
    - `inputSignal` — управляющее воздействие
    - **Возвращает:** температура объекта

  - `void setInitialValue(double initialValue)`
    Задание начальных условий

  - `double getCurrentValue() const`
    Получение текущей температуры объекта
